# Site Avanti

Projeto desenvolvido como parte do processo seletivo do Programa de Estágio Innovation Class - Desenvolvimento de E-commerce da Avanti.

O objetivo foi reproduzir com fidelidade o layout disponibilizado no Figma utilizando HTML, CSS e JavaScript, seguindo as versões Desktop e Mobile.

## Tecnologias utilizadas

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Swiper.js

## Funcionalidades

- Layout responsivo para desktop e mobile
- Menu de categorias com interação em JavaScript
- Carrossel principal com autoplay
- Carrossel de produtos utilizando Swiper
- Menu de navegação interativo
- Newsletter
- Estrutura organizada para facilitar manutenção

## Observações

O projeto foi desenvolvido buscando seguir o layout do Figma da forma mais fiel possível, respeitando cores, tipografia, espaçamentos, componentes e comportamento responsivo propostos no desafio.
